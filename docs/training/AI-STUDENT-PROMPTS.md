# AI Student Prompts: Your Personal Coding Tutor

To get the most out of this training, you should treat the AI Agent as a Senior Mentor. Use these specific prompts to customize your learning journey.

## 1. Learning a New File
If you open a file like `repo-php/class/Router.php` and feel overwhelmed, ask:
> "Explain the logic of `repo-php/class/Router.php` like I'm a beginner. What is the most important method in this file and how does it connect to `public/index.php`?"

## 2. Challenging Yourself
If you finish a task and want to push deeper, ask:
> "I just completed the 'Form Handling' lab in Phase 1. Suggest 3 small features I could add to the `forms` plugin to make it more secure or user-friendly."

## 3. Debugging Help
When you break something and the app shows an error:
> "I modified the `Controller.php` in `repo-php` and now I'm getting a 500 error. Here is the code I changed. Can you find the logic error and explain WHY it failed?"

## 4. Architectural Thinking
When you move to Phase 3, ask:
> "Compare the routing mechanism in `repo-php` (PHP) and `repo-go` (Go) in this project. What are the performance advantages of the Go approach for high-traffic sites?"

## 5. Pair Programming
When working on a lab, ask:
> "I'm about to start the 'Vue 3 Deep Dive' lab. Act as my pair programmer. Walk me through the first step of creating a new view and help me verify that the routing works."

## 6. Real-world Simulation
> "I want to pretend this is a real job. Give me a 'Ticket' or 'Jira Task' describing a bug in the `repo-php` CMS that I need to find and fix today."
