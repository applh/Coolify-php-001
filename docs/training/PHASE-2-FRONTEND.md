# Phase 2: Modern Frontend Mastery (225 Hours)

Transition from static templates to highly interactive, state-driven user interfaces.

## Learning Objectives
- TypeScript fundamentals.
- Component architecture (Props, State, Lifecycle).
- Vite as a build tool.
- Single Page Application (SPA) routing.

## Practical Labs

### 1. Vue 3 Deep Dive (70h)
**Reference**: `repo-vue/src/App.vue` and `src/App.vue`
- **Task**: Compare the main application UI (`src/App.vue`) with the demo app in `repo-vue`.
- **Exercise**: Add a new view in `src/views/` and register it in the router.
- **Goal**: Understand how the dashboard manages the state of various repos.

### 2. React Fundamentals (70h)
**Reference**: `repo-react/src/App.tsx`
- **Task**: Study the simplified React implementation.
- **Exercise**: Rewrite a PHP component from `repo-php/views/components/` into a React function component in `repo-react/src/components/`.

### 3. Design Systems with Tailwind (45h)
**Reference**: `src/style.css`
- **Task**: Learn about `@import "tailwindcss";` and custom themes.
- **Exercise**: Implement a "Dark Mode" toggle for the main dashboard.

### 4. State Synchronization (40h)
**Reference**: `src/views/SyncState.vue`
- **Task**: Investigate how data flows between the backend and frontend.
