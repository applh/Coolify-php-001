# OnePage Demo Documentation

## Overview
**onepage.demo** is a minimalist, single-page portfolio template designed for high performance and clean aesthetics. It follows the "Less is Better" philosophy, focusing on a streamlined user journey.

## Key Features
- **Single-Page Architecture**: All content is served in a single vertical flow with smooth anchor navigation.
- **Modern Stack**: Built with Tailwind CSS and Plus Jakarta Sans typography.
- **Responsive Design**: Mobile-first layout with a sticky blurred navigation bar.
- **Project Showcase**: A grid-based project section with hover interactions.
- **Scroll-Smooth**: Native CSS smooth scrolling enabled for navigation links.

## Structure
- **File**: `repo-php/content/onepage.demo/index.php`
- **Sections**:
    - Hero: "Crafting digital experiences that matter."
    - Work: Grid of 4 selected projects.
    - About: Minimalist mission statement.
    - Footer: Social links and copyright.

## Customization
To update the projects, modify the loop in the `Work` section. The styling can be adjusted directly via Tailwind utility classes in the main `index.php` file.
